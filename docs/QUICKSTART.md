# Быстрый старт

## 1. Установка

```bash
# Запустить установку зависимостей
./setup.sh
```

## 2. Настройка

```bash
# Скопировать пример конфигурации
cp .env.example .env

# Отредактировать конфигурацию
nano .env
```

Заполните следующие параметры в `.env`:

### BOT_TOKEN
Получите у [@BotFather](https://t.me/BotFather) в Telegram:
1. Отправьте `/newbot`
2. Введите имя бота
3. Введите username бота (должен заканчиваться на `bot`)
4. Скопируйте токен

### API_ID и API_HASH
Получите на [my.telegram.org/apps](https://my.telegram.org/apps):
1. Войдите с вашим номером телефона
2. Создайте новое приложение
3. Скопируйте `api_id` и `api_hash`

### OPENROUTER_API_KEY
Получите на [openrouter.ai](https://openrouter.ai/):
1. Зарегистрируйтесь/войдите
2. Перейдите в раздел Keys
3. Создайте новый API key
4. Пополните баланс (минимум $5 рекомендуется)

## 3. Первый запуск

```bash
./run.sh
```

При первом запуске вам нужно авторизовать Telegram аккаунт:
1. Введите номер телефона в международном формате (например: +79991234567)
2. Введите код из Telegram
3. Если включена 2FA - введите пароль

**⚠️ Важно:** Используйте аккаунт, который подписан на каналы, которые хотите отслеживать!

## 4. Использование

Откройте бота в Telegram и отправьте `/start`

### Основные команды:

```
/add @durov              # Добавить канал
/list                    # Показать список каналов
/period                  # Настроить период отправки
/summary                 # Получить саммари сейчас
```

### Пример использования:

```
/add @python_news
/add @tech_channel
/period                  # Выбрать "Раз в день"
/summary                 # Получить саммари
```

## 5. Автоматическая отправка

После настройки периода бот будет автоматически отправлять саммари:
- **Раз в день** - каждые 24 часа
- **Раз в 3 дня** - каждые 72 часа
- **Раз в неделю** - каждые 7 дней

## Остановка бота

Нажмите `Ctrl+C` в терминале

## Запуск в фоне

```bash
screen -S telegram-bot
./run.sh
# Нажать Ctrl+A, затем D

# Вернуться к боту:
screen -r telegram-bot
```
